Règles du jeu :

Les règles du jeu sont simples, aller le plus loin possible sans toucher le moindre tuyau, sinon la destruction est assurée !

Pour jouer à notre jeu, vous avez deux possibilités quand vous êtes sur un ordinateur :

  • Jouer avec les flèches directionnelles depuis votre clavier
  • Jouer avec les flèches directionnelles présentes sur l'écran

Depuis un smartphone, seule la deuxième possibilité est envisageable.
